# Grade Management System

## Developed By
Abhishek Kumar

## Project Description
The Grade Management System is a Java console-based application that manages student marks, calculates averages, assigns grades, and generates performance reports.

## Features
- Add Student Marks
- View Student Records
- Search Student
- Calculate Average Marks
- Grade Categorization
- Performance Reports
- File Handling Support
- Input Validation

## Technologies Used
- Java
- VS Code
- Arrays
- ArrayList
- File Handling
- OOP

## Installation

1. Install JDK 17+
2. Open project in VS Code
3. Compile:

javac *.java

4. Run:

java GradeManagementSystem
